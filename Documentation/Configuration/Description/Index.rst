Description
-----------

Built-in shopping basket and products display within TYPO3. Has a clearing interface which lets you
write your own implementation with existing payment-gateways. 

Currently there's an implementation with DIBS in Denmark, found at `www.architrade.dk
<http://www.architrade.dk/>`__  .
